# GUI Portfolio & React Learning Journey 🚀

Welcome to my personal portfolio! This project serves as a dual-purpose platform: a professional GUI-based showcase of my work and a dedicated space where I document my journey mastering the React ecosystem.

## 📋 Project Overview

This portfolio deviates from traditional scrolling websites by offering a **Graphical User Interface (GUI)** experience. It is designed to be minimalist, interactive, and highly responsive, reflecting my interest in clean code and user-centric design.

## 🎯 Project Purpose
The primary goal of this project is to bridge the gap between academic learning and professional industry standards. It documents my progress as a Junior Developer, specifically focusing on:
- Preparing for professional internship opportunities.
- Demonstrating a "Job-Ready" mindset through clean code and UI/UX best practices.
- Hosting an interactive, downloadable version of my professional Resume.
  
## 🛠️ Tech Stack

*   **Frontend:** [React](https://reactjs.org/) (Functional Components, Hooks)
*   **Build Tool:** [Vite](https://vitejs.dev/) (For lightning-fast development)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Utility-first CSS)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/) (For smooth transitions and scroll reveals)
*   **Language:** JavaScript (ES6+)
*   **Internationalization:** [i18next](https://www.i18next.com/) (Support for English & Myanmar languages)

## ✨ Key Features

- **Auto-hiding Navigation:** A smart navbar that hides during inactivity to provide a distraction-free viewing experience.
- **Scroll-Reveal Animations:** Sections and project cards animate gracefully into view using Framer Motion stagger effects.
- **Theming:** Full support for Light and Dark modes.
- **Bilingual Support:** Easily toggle between English and Myanmar languages.
- **Responsive Design:** Optimized for all screen sizes, from mobile devices to large desktops.
- **Intro Loading Screen:** A custom-timed 2-second system initialization animation for a premium feel.

## 🚀 Learning Roadmap

This project is a milestone in my React learning path. It specifically showcases my implementation of:
- Advanced React Hooks (`useState`, `useRef`,etc...).
- Custom Hooks for window resizing and event handling.
- Complex state management for UI/UX elements.
- Component-based architecture for scalability.
